系统简介

本系统面向企业员工考勤管理场景，按员工端和管理员端划分功能。管理员可维护基本信息、录入各类考勤记录，并进行多条件查询和月度部门统计。

Spring Boot
MyBatis
MySQL
Thymeleaf
