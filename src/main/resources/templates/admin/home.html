系统简介

本系统用于管理员维护员工基本信息、考勤记录、请假记录、加班记录和出差记录，并支持多条件综合查询和月度部门考勤统计。员工端通过工号进入，只能查看和维护自己的联系方式。

Spring Boot
MyBatis XML
MySQL
Thymeleaf
